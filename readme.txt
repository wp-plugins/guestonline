== Description ==

Instabook plugin has been created for Tableonline use.  This plugin build up two widgets which has been designed particularly for Tableonline's users.  

The first widget enables the user to publish the Booking module on their website and the second widget concerns the Contact subscription newsletter module.  Both widget can be displayed either as a button widget or an iframe widget.

== Installation ==

Upload the Instabook plugin to your blog, Activate it, then enter your Referal Key.

You can enter your referal key by either using to link which is beside the desactivate link or by using the Instabook submenu which is found in the Plugin menu.  Both will lead you to the administration page.
